# FaceASIScope
A device to detect facial asymmetry in stroke patients 

This repository contains the software code and hardware designs for the paper "FACIAL SYMMETRY QUANTIFICATION: LANDMARK-BASED ANALYSIS THROUGH COMPUTER VISION" presented at the DMD2026 conference in April 2026.

The FaceASIScope device was built by Anthony Gomez, Natalie Mar and Katherine Zoller as part of the Fall/Spring 2026 Senior Design Capstone Project in the University of Texas at Arlington Bioengineering Department. 

The project was supervised by Dr. DaiWai Olson and Dr. Oguz Yetkin.

The project will be released under the GPLv3 license once complete.  Clinical studies are under way to validate the device.
